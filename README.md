## 核心技巧速查（12 個）

| 技巧 | 適用情境 |
|------|---------|
| 給足 Context | 所有任務 |
| 指定資料來源 | 事實查詢、健康、法律 |
| Deep Research 模式 | 複雜研究問題 |
| Think Hard / Ultra Think | 複雜決策、多文件分析 |
| 多輪迭代反饋 | Brainstorming、寫作 |
| 中立提問 | 任何需要客觀評估的場合 |
| Rubric 評分 | 評估作品、商業提案 |
| Progressive Outlining | 任何長篇文章寫作 |
| 分段編輯 | 修改現有文章 |
| 開新對話 | 切換全新主題時 |
| 圖像 prompt 語言 | 圖像生成 |
| 定期換模型 | 長期 AI 使用習慣 |
